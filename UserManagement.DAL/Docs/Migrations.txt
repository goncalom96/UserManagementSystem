EF6 Migrations + MVC- Criar DAL e Projeto pretendido;- Adicionar package Entity Framework 6 aos projetos;- Definir Models e criar DBContext;- Definir App.config no DAL- Ativar o mecanismo de migrations manual:	1) Selecionar o projeto DAL;	2) Package Manager Console: enable-migrations;		2.1) É criada a pasta Migrations, com o ficheiro Configuration.cs;	3) Package Manager Console: add-migration M01Initial;		3.1) Na pasta Migrations é criado a classe da migration;- Criar Base de dados:	1) Package Manager Console: update-database;		1.1) No SQL Server Object Explorer são criadas as tabelas pretendidas + MigrationHistory;	2) Build para compilar o projeto e gerar a Assembly;- Referenciar DAL ao projeto Web;- Definir App.config no projeto Web;- Executar a aplicação e ver as tabelas;- Testar WebApp para testar DAL (Ex: criar algo);- DAL > fazer alteração estrutural e gerar a respetiva migration:	1) Exemplo: acrescentar ao model uma nova propriedade;	2) Gerar uma nova migration -> Package Manager Console: add-migration M02Book_UpField_DDC (exemplo);	3) Usar a nova migration para atualizar a base de dados: update-database.
